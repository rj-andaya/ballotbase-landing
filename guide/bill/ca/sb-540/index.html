SB 540, explained

Independent System Operator: independent regional organization: California Renewables Portfolio Standard Program.

Active In Committee Process · Author: Becker

In plain English

This bill removes the process that would have allowed California's electrical grid operator (called the ISO) to transform into a regional organization. It deletes the rules and deadlines that were set up for this transformation to happen.

If this passes

What would actually change, according to the bill's official digest. No predictions, no opinions.

  • The process for transforming the ISO into a regional organization would be removed from state law
  • The January 1, 2019 deadline for approvals from the Legislature, Governor, and other state agencies would no longer apply
  • Rules about changing the ISO's governance structure (its bylaws and corporate documents) as part of this transformation would be deleted
