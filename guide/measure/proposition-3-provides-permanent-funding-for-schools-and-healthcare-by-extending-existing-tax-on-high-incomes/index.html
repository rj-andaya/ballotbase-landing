PROPOSITION 3 PROVIDES PERMANENT FUNDING FOR SCHOOLS AND HEALTHCARE BY EXTENDING EXISTING TAX ON HIGH INCOMES

In plain English

This measure would continue a tax on high incomes that is currently set to expire. The money collected would go to schools and healthcare. If it passes, this tax would stay in place permanently instead of ending.
