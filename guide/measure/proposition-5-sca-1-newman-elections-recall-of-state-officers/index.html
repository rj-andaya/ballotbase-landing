PROPOSITION 5 SCA 1 (Newman) Elections: recall of state officers

On the ballot CA · 2026-11-03

In plain English

This measure would change the rules for recalling state officials in California. It would lower the number of voter signatures needed to start a recall election for the Governor, state legislators, and other state officers.

If this passes

What would actually change. No predictions, no opinions.

  • Recall petitions for state officers would require fewer signatures to qualify for an election
  • The signature requirement would be lowered from the current level, making it easier to trigger a recall vote
  • State officials including the Governor and state legislators would be subject to these new, lower signature thresholds
  • Voters would have an easier path to force a recall election for state-level positions
