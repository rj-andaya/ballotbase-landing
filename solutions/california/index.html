The problem

Cal-Access is notoriously difficult to use

  • ✕ Opaque interface designed for regulators, not researchers
  • ✕ Data spread across dozens of tables with inconsistent formatting
  • ✕ Amendment filings create duplicate rows without clear audit trails
  • ✕ No built-in analytics — just raw transaction exports
  • ✕ Schedule D and F496 data inflate totals if naively summed
The BallotBase solution

We do the hard work for you

  • ✓ Clean, visual dashboards built directly on Cal-Access data
  • ✓ Amendment deduplication — only the latest filing version counts
  • ✓ Correct totals: Schedule E only for expenditures, F460 only for receipts
  • ✓ Automatic committee type detection — candidate vs. IE
  • ✓ Enter a filer ID and your dashboard is ready in seconds
